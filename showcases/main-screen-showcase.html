Design Brief — Main Screen

A floating, keyboard-centric overlay that replaces (or complements) the menubar panel as the primary way to interact with Cortado. Opened via a global hotkey (⌘+Shift+K or similar), navigated entirely with arrow keys, dismissed with Esc. Shows feed activities with status at a glance. Enter opens the selected activity. Must feel instant, native, and glanceable.

Five high-variance alternatives explore different interaction models, information density, and layout shapes. All use realistic feed data from the spec.

Fidelity Notes
  • These are static mockups — keyboard navigation is indicated visually (blue outlines) but not wired.
  • Native vibrancy/blur is approximated with backdrop-filter; real app uses NSPanel translucency.
  • Panel sizing and shadow depth are representative but will need tuning in the real window context.
  • Data shown matches the FeedSnapshot / Activity / Field types from the app.

A · Spotlight — Search-first command palette

Horizontal bar centered in the upper third of the screen, like macOS Spotlight or Raycast. A search/filter input sits at top. Activities are listed flat below, lightly grouped by feed. Type to filter, arrows to navigate, Enter to open. Extremely fast for users who know what they're looking for.

Search-first Flat list Wide · short Filter-driven Familiar paradigm

B · Columns — Multi-feed overview

Wider centered panel with feeds as side-by-side columns, like a kanban board or multi-pane dashboard. Each column shows its feed's activities compactly. Navigate with Left/Right between feeds, Up/Down within. Best when you want to see the full picture across all feeds simultaneously.

Multi-column All feeds visible Wide · tall 2D navigation Overview-first

C · Priority — Severity-ranked flat list

Narrow panel that ignores feed grouping entirely. Activities are sorted by status severity — attention-negative items float to the top, idle/retained at the bottom. Grouped by status kind, each with a tinted label. The most action-oriented view: what needs my attention right now?

Priority-sorted Status-grouped Narrow · focused Action-oriented No feed grouping

D · Split — Master-detail pane

Two-column floating panel. Left pane is a compact activity list grouped by feed. Right pane shows full field details for the selected activity, plus an "Open" action. Navigate the list with Up/Down; Right arrow moves focus to detail; Enter opens. Best for power users who want full context without leaving the overlay.

Master-detail Full field info Wide · balanced Two-zone navigation Power-user friendly

E · Dense Rows — Compact table with feed color coding

Medium-width panel with a flat table-like list. Each row has a thin left-edge bar color-coded to its feed, a status dot, the title, an inline status value, and a timestamp. No grouping headers — the color bar handles feed identification. Maximum information density in minimum vertical space.

Table-like Color-coded feeds High density Inline status Minimal chrome
