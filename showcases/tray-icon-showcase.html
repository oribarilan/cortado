Cortado Tray Icon

Menubar icon variations for Cortado. Each is a macOS template image — the OS tints it automatically (dark on light bar, light on dark bar). Shown at actual size (18px within 22pt) and enlarged for detail.

Variants

Six directions for the cortado glass silhouette. Each row simulates macOS light & dark menu bars.

How template images work: macOS ignores the RGB channels entirely and uses only the alpha channel. Opaque pixels get tinted to match the menu bar style; transparent pixels are invisible. The white "cut" lines in these SVGs create transparent gaps in the silhouette, forming the espresso/milk layer line — the signature cortado look.

Sizing: macOS tray icons are 22x22 points (44x44 px on Retina). The icon art itself is typically ~18px tall with 2px padding. These previews show the art at 18px (actual), 30px (2x), and 56px (detail).
