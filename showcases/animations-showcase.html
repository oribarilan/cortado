Sprint 11 — Animations & Transitions

Purposeful motion throughout the app. Subtle, fast animations that make the UI feel responsive and alive without drawing attention to themselves. Every animation respects prefers-reduced-motion.

Animation Tokens

01-animation-tokens

Shared CSS custom properties for durations and easings. These replace all hardcoded transition values across the three windows (menubar panel, main screen, settings) and automatically disable under prefers-reduced-motion.

Durations

--duration-fast 100ms
--duration-normal 180ms
--duration-slow 280ms

Easings

--ease-out cubic-bezier(0.22, 1, 0.36, 1)
--ease-in-out cubic-bezier(0.65, 0, 0.35, 1)
