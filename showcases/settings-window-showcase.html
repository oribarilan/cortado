Cortado Settings — Design Showcase

Four distinct approaches to the settings window. Each variant explores different layout, typography, and tone — from precision instruments to warm editorial.

A — Swiss Instrument

Sidebar + Clean Grid

Precision-oriented. Sidebar navigation with a clinical, high-contrast layout. Instrument Sans typography gives it a contemporary European feel. Tight spacing, clear hierarchy. The "serious tool" option.

B — Editorial

Top Tabs + Serif Headings

Refined and warm. Pairs DM Sans body text with Source Serif headings for an editorial quality. Top-mounted tabs, generous whitespace. The "considered design" option — feels more opinionated than a typical settings panel.

C — Utilitarian Dense

Dark + Table Layout

Developer-first density. Dark theme, compact table view for feeds, Space Grotesk typography. Feels like a terminal-adjacent control panel. Teal accent for a technical but not cold personality. The "power user" option.

D — Warm Studio

Cards + Serif Display

Approachable and warm. Fraunces serif headings add personality, Outfit body text stays clean. Feeds presented as cards with color-coded type badges. Warm neutral tones (hue 80) instead of blue-gray. The "friendly craft tool" option.
