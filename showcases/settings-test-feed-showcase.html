Test Feed — Results Display

Three ways to show test results when the user clicks "Test" on an unsaved feed form. Each shown in loading, success (with activity preview), and error states.

T1 — Inline Panel

Recommended

A result panel appears below the "Test" button. Shows a spinner while polling, then expands to show status + activity preview on success, or error message on failure. Dismissible with ✕. The form stays fully visible above.

T2 — Compact Toast

Minimal

A one-line toast banner appears below the buttons. Success shows count only ("✓ 3 activities found"), error shows the message. No activity preview. Simplest to build. Click to expand details optionally.

T3 — Panel + Collapsible Preview

Progressive

Like T1 but the activity preview is collapsed by default behind a "Show preview" toggle. The status line is always visible ("✓ Connected — 3 activities"). Keeps the form compact by default but lets curious users peek at what the feed returned.
