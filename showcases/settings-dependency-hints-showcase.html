Dependency Hints — Options

Four ways to communicate that a feed type requires an external CLI tool. Each shown for all three feed types (github-pr, ado-pr, shell).

D1 — Inline Hint

Contextual

A small colored banner appears directly below the type selector. Changes dynamically when the type changes. Compact, impossible to miss, doesn't take much space. Link opens the install page.

D3 — Collapsible Prerequisites

Progressive

A collapsible "Prerequisites" section below the type selector. Collapsed by default — just a header "Prerequisites ▸". Expanded, it shows install steps, auth commands, and links. Non-intrusive for experienced users, detailed for newcomers.

D4 — Footer Note

Subtle

A small note at the bottom of the form, above the Save/Discard buttons. Minimal visual weight — just text and a link. Trusts the developer to notice it. The most "quiet" option.
