L5 — Sidebar + Cards

Same sidebar navigation, four different card treatments for feeds. Varying info density, visual weight, and spatial feel.

L5a — Compact Single-Line

Tight / Inline Meta

Cards reduced to their minimum — one line, left-edge indicator, name + type + interval inline. Almost as dense as a table row but with card affordance. Best for many feeds.

Standard / Rich Info

L5c — Two-Column Grid

Spatial / Grid

Cards laid out in a 2-column grid. Each feed is a self-contained tile with type badge and metadata. More spatial, more visual. Includes an "Add feed" placeholder tile. Works best when you want each feed to feel like its own entity.

L5d — Borderless Elevation

Soft / No Borders

Cards differentiated by background elevation, not borders. Transparent border at rest; on hover, a subtle border and shadow appear. Status dot instead of edge indicator. Softer, less "boxy" — closer in spirit to the menubar panel's own row style.
