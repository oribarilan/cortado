Sidebar + Table

The original C, polished. Sidebar navigation, tabular feed list, teal accent. Clean separation between nav and content. Most conventional settings layout — instantly familiar.

C2 — Amber Accent

Sidebar + Flat List

Stripped to essentials. No table chrome — feeds are a flat hover-to-reveal list. Amber/gold accent adds warmth without sacrificing the technical feel. Smaller toggles, tighter spacing. The most minimal option.

C3 — Violet Accent

Segmented Nav + Cards

Sidebar replaced by a segmented control (pill switcher) in a top bar. Feeds shown as compact cards with color-coded left-edge indicators. Hover reveals action buttons. Violet accent gives it a distinct identity. The most "app-like" of the four.

C4 — Blue Accent

No Sidebar / Sections

No sidebar, no tabs — a single scrollable column with collapsible sections. Blue accent matches the panel's existing blue-gray identity for maximum cohesion. Feeds shown as a compact flat list with status dots. Feels like a focused config document — closest to how a TOML file already reads.
