Recommendation: B (Subtle Left Border). It keeps the header visually consistent with feed headers (same color, same weight, same case) while adding a single structural cue that is unmistakable but not loud. The border uses a visual language distinct from the text -- it doesn't compete with the row-level status colors. The code change is minimal (one CSS property, one text change, remove the icon).

Runner-up: C (Muted Warm Tint). Even simpler code change, and the desaturated warmth is beautifully subtle. The risk is that it's too subtle -- the hue shift may be imperceptible on some displays, especially in light mode. If chosen, consider pairing it with a slightly wider letter-spacing or a new --text-tertiary-warm token to make the intent explicit.
