# Sprint 10 -- Hotkeys

## Theme
Keyboard shortcuts: global hotkey to open/focus the app, and in-app navigation hotkeys.

## Tasks

1. **j/k navigation** — Add vim-style j/k as aliases for ↑/↓ in the panel.
2. **Configurable global hotkey** — Make the panel toggle hotkey (currently ⌘⇧Space) configurable from Settings > General with a keyboard recorder widget. Flows: record, escape-to-cancel, clear.

## Sequencing
Tasks 1 and 2 are independent. Task 2 spans Rust model, startup, command, and frontend UI.
